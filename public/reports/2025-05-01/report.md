# Тестовый новостной макет — 2025-05-01

Это демонстрационный пример будущего HTML-формата для раздела новостей.

- Взяты только 3 раздела: Hacker News, GitHub Trending, Latent Space AINews.
- Все заголовки переведены на русский.
- Каждый элемент — это ссылка на оригинал с коротким нейтральным подзаголовком.
- Метаданные под заголовками намеренно отсутствуют.
- Это не полноценный выпуск, а тестовая витрина структуры.
